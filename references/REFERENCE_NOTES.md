# Reference notes

This git-ready package does not redistribute the uploaded PDFs. The proposal is based on two papers:

1. Miconi & Kay, "Neural mechanisms of relational learning and fast knowledge reassembly in plastic neural networks". This paper motivates the meta-training plastic RNN, Hebbian plasticity, neuromodulation, active/passive solutions, and recoded reinstatement.
2. Liu, Wang & Luo, "Human brains construct individualized global rankings from identical few-shot learning input". This paper provides the target behavior: identical few-shot local input leads to stable, self-consistent, idiosyncratic global rankings.
